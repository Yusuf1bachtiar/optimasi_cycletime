Langkah-langkah dalam Proses Optimasi Cycle Time:

  1. Pengumpulan Data: Data dari mesin dan karyawan dikumpulkan untuk analisis lebih lanjut.
  2. Preprocessing Data: Data yang dikumpulkan dibersihkan dan disiapkan untuk model Naive Bayes.
  3. Pemilihan Fitur: Fitur-fitur yang relevan dipilih untuk membangun model prediksi yang akurat.
  4. Pemilihan Model: Model Naive Bayes dipilih untuk memprediksi cycle time berdasarkan data input.
  5. Pelatihan Model: Model Naive Bayes dilatih dengan data yang sudah diproses.
  6. Prediksi: Model yang terlatih digunakan untuk memprediksi cycle time berdasarkan input data.
  7. Evaluasi Output: Hasil prediksi dievaluasi untuk menilai kinerja model dan melakukan penyesuaian jika perlu.
